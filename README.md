# CNPX Soat

## Background:
The task is to create a simple web application to buy vehicles insured policies using Ruby on Rails

## Requirements:
```
Ruby 2.4.0
Rails 5.0.2
```

## Bundle:
```
  bundle install
```

## DB
```
  rake db:create db:migrate db:seed
```

## RUN
```
  rails s
```

## RSPEC
```
  rspec
```
